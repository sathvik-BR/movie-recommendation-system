# 🎬 Movie Recommendation System

A Machine Learning based web application that recommends similar movies using content-based filtering.

## 🚀 Features
- Recommends movies based on similarity
- Uses TMDB 5000 Movie Dataset
- Interactive web interface using Streamlit
- Machine learning similarity model

## 🛠 Technologies Used
- Python
- Pandas
- Scikit-Learn
- Cosine Similarity
- Streamlit

## 📂 Dataset
TMDB 5000 Movie Dataset

## ▶ How to Run

1. Install required libraries
pip install pandas scikit-learn streamlit

2. Run the application
streamlit run app.py

## 📸 Demo
Select a movie and click **Recommend** to get similar movies.
